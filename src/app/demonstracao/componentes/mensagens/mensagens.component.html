As notificações são chamadas pelo Toastr Service, da biblioteca Ngx-Toastr.
       

O "spinner" de carregamento pode ser chamado pelo NgxSpinnerService, da biblioteca Ngx Spinner.

Um modal pode ser chamado pelo serviço ModalService, com várias opções de customização.
Notificações Spinner Modal
Notificações (Toastr)

Biblioteca Ngx-Toastr

No módulo: 
import {{ '{' }} ToastrModule  {{ '}' }} from 'ngx-toastr';
No componente: 
import {{ '{' }} ToastrService {{ '}' }} from 'ngx-toastr';

Spinner (NgxSpinner)

Biblioteca NgxSpinner

No AppModule: 
import {{ '{' }} NgxSpinnerModule  {{ '}' }} from 'ngx-toastr';
No AppComponent (.ts): 
import {{ '{' }} NgxSpinnerService {{ '}' }} from 'ngx-spinner';
No AppComponent (.html): 
<cx-spinner [fullscreen]=true></cx-spinner>

Modal
No AppComponent (.html): 
<cx-modal></cx-modal>
No componente: 
import {{ '{' }} ModalService {{ '}' }} from 'src/app/guia-caixa/services/modal.service';

show() Exibir o modal

titulo: string
mensagem: string

btCancelarTexto: string

classTitulo: string
btnOkClass: string
btnCancelarClass: string
modalBodyClass: string
modalHeaderClass: string
modalFooterClass: string
centralizado: boolean
tamanho: ModalSize
