A diretiva inputCaixa utilizada em conjunto com o componente <cx-input> permite a estilização e validação automática do campo.

O componente exibirá mensagens predefinidas para cada um dos "Validators" do Angular Forms.

Validadores customizados podem ser utilizados passando-os como argumento pela propriedade customErrors, como um objeto ou um Array de objetos no formato:
{ "nomeDoErro": "Mensagem a ser exibida" }.
Inputs básicos Validação Validação Customizada Ícones Classe
InputCaixa

Diretiva que permite a interação entre os elementos input e textarea com o componente cx-input.

Usar como: inputCaixa


Cx-Input

Componente que engloba o campo com a diretiva inputCaixa, e exibe os ícones e mensagens de feedback.

Usar como: inputCaixa

Propriedades
Nome Descrição
@Input( )
showFeedbackIcon: boolean		 Exibir ícone de feedback quando campo está válido ou inválido.
Padrão: true
@Input( )
showFeedbackMessage: boolean		 Exibir mensagem de erro abaixo do campo quando está inválido.
Padrão: true
@Input( )
customErrors: string | any[ ]		 Validators personalizados que possuem suas próprias mensagens de feedback.
Utiliza-se um array de objetos, onde cada validator deve ser representado como: 
{{ '{' }} "nomeDoValidator": "Mensagem de erro do Validator" {{ '}' }}
Padrão: true
@Input( )
formato: string		 Campo opcional para sugerir ao usuário o formato necessário para o campo.
Diferentemente do atributo placeholder, o formato será exibido abaixo do campo.
Pode-se exibir ícones utilizando a classe prefix-icon em qualquer elemento.

Uma versão mais simples, sem a validação automática, pode ser utilizada apenas com a classe input-caixa ou, para diferentes temas, input-caixa-(tema).
